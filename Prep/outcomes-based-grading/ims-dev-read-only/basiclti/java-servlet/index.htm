IMS BasicLTI Java Sample Code

This is sample code which to support the IMS BasicLTI and IMS Common Cartridge Specifications. There are examples of the following in this source tree:

  • How the Learning Management System authors and launches a BasicLTI resource. (i.e. a BasicLTI Consumer)
  • An external tool which supports BasicLTI (i.e. a BasicLTI Provider) - Note that this tool will fail when launched with a GET request. It requires a POST request with propertly signed OAuth data.

Note: Unpublished drafts of IMS Specifications are only available to IMS members and any software based on an unpublished draft is subject to change. Sample code is provided to help developers understand the specification more quickly. Simply interoperating with this or any sample implementation code does not allow one to claim compliance with a specification.
